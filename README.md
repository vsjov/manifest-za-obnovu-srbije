# Manifest za obnovu Srbije

**Manifest za obnovu Srbije** je građanski i nestranački dokument koji analizira razgradnju institucija, korupciju, zarobljavanje države, medijsku propagandu, odliv ljudi i poništavanje demokratskog napretka u Srbiji. Dokument predlaže jedanaest načela za obnovu države: suverenitet institucija, radikalnu transparentnost, obrazovanje, ekonomsko dostojanstvo, slobodu štampe, građansko učešće, meritokratiju, ograničenje koncentracije vlasti, kulturni preporod, povratak dijaspore i razdvajanje crkve i države.

Ovaj manifest ne pripada nijednoj stranci, vođi ili pokretu. Namenjen je građanima koji veruju da Srbija može biti slobodnija, pravednija, odgovornija i institucionalno zdravija država.
