## Security Considerations

This is a static site generator that processes templates at build time. Since all content comes from source-controlled files, there are no runtime security concerns like XSS. The built HTML is static and safe to deploy.

**Note:** If you extend the build process to fetch data from external APIs, ensure those sources are trusted.
